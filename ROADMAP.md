# Chat Buddy Roadmap

This roadmap outlines the planned features and long-term vision for Chat Buddy. Our goal is to create the most powerful, flexible, and personal AI assistant accessible directly from your favorite messaging platforms.

## Planned Features

- [ ] **Multi-LLM Support**: Integrate support for Google Gemini, Anthropic Claude, and Ollama for running private, local models.
- [ ] **Telegram Integration**: Expand support beyond WhatsApp to allow Chat Buddy to operate seamlessly on Telegram.
- [ ] **Persistent Memory via Vector Database**: Upgrade short-term RAM memory to persistent, long-term memory utilizing a vector database for rich context recall.
- [ ] **Voice Message Transcription**: Allow the bot to listen to incoming voice notes, transcribe them automatically, and reply contextually.
- [ ] **Web Dashboard**: Introduce a graphical interface to monitor active sessions, manage configurations, and review bot metrics.
- [ ] **Plugin System for Custom Tools**: Develop a modular plugin architecture so developers can easily create and share custom tools for the agent to use.
- [ ] **Rate Limiting per User**: Implement safeguards to prevent abuse by setting message quotas and rate limits per user.
- [ ] **Multi-Language Reply Support**: Enhance the bot's capabilities to detect user languages dynamically and respond natively.

_Note: This roadmap is a living document and priorities may shift based on community feedback and contributions._
