# Support

Need help using Chat Buddy or have a question? We're here for you!

## Community Support

For general questions, troubleshooting, setup issues, or discussing ideas, please engage with the community:

- **[Open a GitHub Discussion](https://github.com/snackoverflowasad/chat-buddy/discussions)**: This is the best place to ask for help so others can learn from the answers too!
- **[Search existing Issues](https://github.com/snackoverflowasad/chat-buddy/issues)**: Your problem might have already been solved.

## Private or Security Concerns

If you have a private issue or have discovered a security vulnerability (such as a flaw related to API keys, WhatsApp sessions, or encryption), **do not open a public issue**.

Instead, responsibly disclose it by emailing:
**techie.asad.dev@gmail.com**

We take security and privacy very seriously and will respond to your email as soon as possible.
